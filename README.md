# MCUdev 🚀

Welcome to **MCUdev**, a project dedicated to designing and developing a custom Microcontroller Unit (MCU) from scratch. The goal is to implement and run this architecture on an FPGA.

## 🛠️ Development & Simulation

The processor architecture is currently being designed and simulated using **Digital**, an easy-to-use digital logic simulator.

* **Simulator Used:** Digital (by hneemann)
* **Repository Link:** [hneemann/digital](https://github.com/hneemann/digital)

---

## 📌 Project Roadmap

- [ ] Design the Instruction Set Architecture (ISA)
- [ ] Implement the Arithmetic Logic Unit (ALU) and Control Unit
- [ ] Simulate and test the core logic in *Digital*
- [ ] Translate the design into a hardware description language (Verilog/VHDL)
- [ ] Deploy and test on an FPGA board
